# Python Backgammon

A GUI-based Backgammon game built with Python and Tkinter.

## Features
- PvP and Player vs AI modes.
- Save/Load game functionality.
- Undo move capability.
- Doubling cube rules.
- Special win rules.

## How to Run
First:
1. Ensure you have Python installed.
2. Clone the repository on you local machine.
   
Option 1:
Run the main script:
  ```bash
  python main.py
  ```

Option 2:
1. Run:
  ```bash
  pip install pyinstaller
  pyinstaller main.py --onefile --noconsole
  ```
This will create an executable file so you can just run that file to play the game.
